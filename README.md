# Котячий квест

Невелика текстова гра із котиком-авантюристом: просинаючись голодним, герой обирає, куди вирушити в пошуках їжі — у ліс, провулок, на дах чи в сусідський сад. Кожен вибір веде до нових ситуацій: успішна риболовля, отруйні ягоди, несподівана зустріч із щуром або вовком, поразка чи перемога. Після завершення гри раунд можна почати з початку, при цьому враховується кількість зіграних ігор для кожного імені.

## Технічні деталі

- **Java Servlets + JSP + JSTL** під Tomcat 9  
- **Maven** для збірки WAR  
- **Сесії HTTP** зберігають ім’я гравця, поточне питання та статистику  
- **JUnit 5 + Mockito** для юніт-тестів логіки і сервлетів
